Problem: PE 969 - Kangaroo hopping

Correct answer (mod 1e9+7): 412543690

Correct approach:
- Let N_n be the first hop count with S_k = X_1+...+X_k > n, where X_i ~ Uniform(0,1) i.i.d.
- Use tail-sum expectation:
  H(n) = E[N_n] = sum_{k>=0} P(S_k <= n).
- Use Irwin-Hall CDF (integer n):
  P(S_k <= n) = (1/k!) * sum_{j=0}^n (-1)^j * C(k,j) * (n-j)^k.
- Swap sums and simplify the inner exponential series:
  H(n) = sum_{j=0}^n (-1)^j * (n-j)^j * e^{n-j} / j!
       = sum_{j=0}^n (-1)^j * (n-j)^j * alpha^{n-j} / j!,
  where alpha = H(1) = e.
- So coefficient of alpha^{n-j} is
  c_{n,j} = (-1)^j * (n-j)^j / j!.
- Integer-coefficient condition:
  j! divides (n-j)^j.
  This is equivalent to (n-j) being divisible by primorial(j) = product of primes <= j.
- Therefore:
  S(n) = sum over j (0<=j<=n, primorial(j) | (n-j)) of (-1)^j * (n-j)^j / j!.
- For total T(N)=sum_{n=1}^N S(n), swap order with n = j + primorial(j)*t:
  T(N) = sum_{j>=0} (-1)^j * primorial(j)^j / j! * sum_{t=1}^{floor((N-j)/primorial(j))} t^j.
- For N=10^18, only j<=52 contribute (for j>=53, floor((N-j)/primorial(j))=0).
- Compute power sums mod 1e9+7 via polynomial evaluation (e.g., Lagrange interpolation).

Wrong approaches and why they fail:
- Guessing a simple closed form like H(n)=alpha^n/n! is false (does not match given H(3)).
- Using only observed sign/parity patterns of S(n) (odd negative, even often positive) is not sufficient; large spikes come from higher-j divisibility events.
- Truncating j by a rough asymptotic estimate alone can be wrong; exact condition floor((N-j)/primorial(j))>=1 must be checked (j=48..52 still contribute for N=10^18).
